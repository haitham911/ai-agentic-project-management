Project: AI-Powered Agentic Workflow for Project Management

1) The Agentic Toolkit - Agent Implementation

- Implement required agent classes with correct structure and initialization.
  - Define these six student-implemented classes in workflow_agents/base_agents.py:
    - DirectPromptAgent
    - AugmentedPromptAgent
    - KnowledgeAugmentedPromptAgent
    - EvaluationAgent
    - RoutingAgent
    - ActionPlanningAgent
  - Each class should have an __init__ method that initializes the needed attributes.
  - Each class should expose its main public method:
    - respond() for prompt-based agents
    - evaluate() for EvaluationAgent
    - route() for RoutingAgent
    - extract_steps_from_prompt() for ActionPlanningAgent
  - Each method should return output in the expected format.

- Configure agents for correct LLM interaction and API key management.
  - Use gpt-3.5-turbo for agents that call the LLM, unless otherwise specified.
  - Use text-embedding-3-large for RoutingAgent embeddings.
  - Pass the OpenAI API key during initialization.
  - Do not hardcode the API key.
  - EvaluationAgent should use temperature=0 for evaluation and correction calls.

- Implement agent-specific prompt engineering and core logic as required.
  - DirectPromptAgent:
    - Sends the user prompt directly as a user message
    - No system prompt
    - Returns only text content
  - AugmentedPromptAgent:
    - Uses a system prompt for persona
    - Instructs the LLM to forget previous context
    - Returns only text content
  - KnowledgeAugmentedPromptAgent:
    - Uses a system prompt for persona and provided knowledge
    - Instructs the LLM to use only that knowledge
    - Returns only text content
  - EvaluationAgent:
    - Runs an iterative loop up to max_interactions
    - Gets a response from a worker agent
    - Evaluates it against criteria
    - If needed, generates correction instructions
    - Returns a dictionary with final response, evaluation, and iteration count
  - RoutingAgent:
    - Gets text embeddings using text-embedding-3-large
    - Computes cosine similarity
    - Selects the best agent
    - Calls its function and returns the selected agent’s response
  - ActionPlanningAgent:
    - Uses a system prompt to extract actionable steps from a user prompt
    - Returns a clean list of steps
